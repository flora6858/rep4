# Random Quote Generator 
**Автор:** [Сулейманов Денис]
**Язык:** Python 3.x
## Описание
GUI-приложение «Random Quote Generator» с использованием random, сохранением истории и Git.
## Возможности
*Создаетсясписок предопределённых цитат (текст, автор, тема).
* Можно сгенерировать цитату», которая выбирает случайную цитату.
* Отображается история сгенерированных цитат в виде списка.
* Реализуется фильтрация: По автору, По теме
